Algorithmie

Algorithme

Un algorithme est une suite d'instructions logiques qui, pour résoudre un problème donné, va intégrer les notions d'entrées et de sorties afin de résoudre le dit problème.

IDE

Un IDE (integrated development environment) est un logiciel de création d'applications, qui rassemble des outils de développement fréquemment utilisés dans une seule interface utilisateur graphique.

Programme

Un programme est une succession d'algorithmes.

Pas à pas

Le pas à pas est un outil qui permet de débugger son code.
