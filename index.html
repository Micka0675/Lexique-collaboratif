Bienvenue dans le Lexique Collaboratif de PhoeniXML! Ce lexique a pour objectif de referencer les informations importantes qui concernent la formation Concepteur Developpeur d'Applications (CDA).

Comment contribuer au projet?

  • Télécharger Github Desktop sur le site officiel
  • Télécharger Visual Studio Code sur le site officiel
  • Ouvrir Github Desktop, connecter le compte
  • Aller dans "cloner un repertoire", selectionner "Lexique-collaboratif"
  • Cliquer sur "Ouvrir le projet dans Vs Code"

A chaque session de modification du projet:

  • !IMPORTANT! Verifier via le bouton "Fetch Origin" que la version du projet en Local est bien la version la plus recente du projet.
  • Effectuer les modificationss voulues depuis VS Code, telecharger l'extension "live server" (facultatif) pour plus de confort de developpement.
  • Une fois les modifs terminées, cliquer sur "Commit to main" (en bas à gauche de l'ecran sur Github Desktop) après avoir donné un nom aux modifs et les avoir décrites."
  • Cliquer sur "Push origin"
  • Bien joué! Modifications effectuées!
